# Bash-Fundamentals

# What is bash?

1. Bash is a command line interpreter as well as a scripting language.

2. Bash is the default scripting language for many operating systems such as:
	-linux
	-mac

3. BASH stands for Bourne Again Shell


# How is this repo helping me learn and practice bash?

2. I can view my previous commits to see my progress.

3. I can review commits where I fixed errors to see how they were fixed.
